# Week 6: Introduction to Python for Data Analysis

Python is one of the most versatile high-level programming languages, widely recognized for its ease of use and robust library ecosystem, specifically **Pandas**.

---

## 📅 Day 1: Core Fundamentals
The first day was dedicated to the building blocks of Python syntax and best practices.

### Basic Skills & Syntax
* **Data Types:** | Type | Description |
    | :--- | :--- |
    | `String` | Textual data |
    | `Float` | Decimal numbers |
    | `Integer` | Whole numbers |
    | `Boolean` | True/False values |
* **Variables:** Learned to use `snake_case` (lower case) and handle user input via `input()`.
* **Escape Characters:** Used to format complex strings and printouts.
* **Mathematical Operators:** Standard arithmetic and logic.

### PEP 8 & Data Structures
We familiarized ourselves with **PEP 8**, the official Python style guide, to ensure code legibility. We also compared the three primary collection types:
1.  **Lists:** Ordered and mutable.
2.  **Tuples:** Ordered and immutable.
3.  **Dictionaries:** Key-value pairs for mapped data.

---

## 🔄 Day 2: Control Flow
Day two shifted focus to shortcut operators and conditional logic.

### Conditional Statements
We explored the three main pillars of flow control:
* **If/Else:** Branching logic based on specific conditions.
* **While Loops:** Executing code while a condition remains true.
* **For Loops:** Iterating over sequences.

> [!TIP]
> **The Fizz-Buzz Challenge:** We utilized these loops to solve the classic Fizz-Buzz challenge, a great way to practice modulo operators and nested logic.

---

## 🐼 Day 3: Data Manipulation with Pandas
We began working with **Pandas**, the industry-standard library for data sets. Its power lies in its ability to analyze, clean, and manipulate data effortlessly.

* **Setup:** `import pandas as pd`
* **DataFrames:** Learned to define, index, and slice DataFrames.
* **Operations:** Practiced advanced data exploration and querying techniques.
